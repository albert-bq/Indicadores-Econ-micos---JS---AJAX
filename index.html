Indicadores Económicos - Guía de Uso

Introducción

Este documento proporciona instrucciones para integrar el script que obtiene indicadores económicos de Chile en tu sitio web. El script utiliza jQuery y AJAX para recuperar los datos.

Lista de Indicadores

A continuación se muestra la lista de indicadores disponibles y sus nombres en código que puedes utilizar:

  • dolar - Dólar observado
  • euro - Euro observado
  • uf - Unidad de Fomento (UF)
  • ivp - Índice de Valor Promedio (IVP)
  • utm - Unidad Tributaria Mensual (UTM)
  • libra_cobre - Libra de cobre
  • tasa_cambio - Tasa de cambio

Instrucciones de Uso

Sigue estos pasos para utilizar el script:

  1. Asegúrate de que jQuery esté incluido en tu página. Puedes añadirlo mediante el siguiente enlace CDN:
    2. <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
  2. Agrega el siguiente contenedor <div> en tu HTML donde desees mostrar el valor del indicador:
    3. <div id="indicador-valor"></div>
  3. Incorpora el siguiente código JavaScript para obtener y mostrar el indicador:
    4. 
// Espera a que el documento esté completamente cargado
$(document).ready(function(){
    // URL de la API que proporciona los indicadores económicos
    const api_url = "https://mindicador.cl/api";

    // Función para obtener el valor de un indicador
    function getIndicador(indicador, callback) {
        // Realiza una solicitud GET a la API
        $.get(api_url, function(data){
            // Verifica si el indicador existe en la respuesta
            if (data[indicador]) {
                // Llama a la función de callback con el valor y la fecha del indicador
                callback(data[indicador].valor, data[indicador].fecha);
            } else {
                // Si el indicador no se encuentra, llama a la función de callback con valores nulos
                callback(null, null);
            }
        });
    }

    // Ejemplo de uso de la función getIndicador
    getIndicador('dolar', function(valor, fecha) {
        // Muestra el valor del indicador en el contenedor con id "indicador-valor"
        if (valor) {
            $('#indicador-valor').text(`Valor del Dólar: ${valor} (Fecha: ${fecha})`);
        } else {
            $('#indicador-valor').text('Indicador no encontrado');
        }
    });

    // Ejemplo de cálculo con el valor de un indicador
    getIndicador('uf', function(valor, fecha) {
        if (valor) {
            const resultado = valor * 2; // Ejemplo de cálculo: multiplicar el valor de la UF por 2
            $('#indicador-valor').append(`
    Valor de la UF: ${valor} (Fecha: ${fecha}) - Ejemplo de Cálculo: ${resultado}
    `);
        }
    });
});

Descripción del Código

A continuación se describe el código JavaScript:

  • Inclusión de jQuery: Incluye jQuery para que el script funcione.
  • Elemento HTML: Añade un <div> con id "indicador-valor" para mostrar el valor del indicador.
  • Función getIndicador: Realiza una solicitud a la API y pasa el valor del indicador a una función de callback.
  • Uso del Script: Muestra un ejemplo de cómo obtener y mostrar el valor del dólar.
  • Ejemplo de Cálculo: Muestra cómo realizar cálculos con el valor de un indicador (en este caso, multiplicar el valor de la UF por 2).

Personalización

Para obtener otros indicadores, cambia el parámetro en la llamada a getIndicador:


getIndicador('euro', function(valor, fecha) {
    // Muestra el valor del indicador Euro
    if (valor) {
        const resultado = valor * 1.1; // Ejemplo de cálculo: multiplicar el valor del Euro por 1.1
        $('#indicador-valor').append(`
Valor del Euro: ${valor} (Fecha: ${fecha}) - Ejemplo de Cálculo: ${resultado}
`);
    }
});

Conclusión

Este script es una forma fácil de integrar datos económicos en tu sitio web y realizar cálculos con ellos. Puedes adaptarlo según tus necesidades.
